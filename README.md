# AgriSack - Premium Industrial Packaging Solutions

AgriSack by Gopal Polytech is your trusted partner for high-quality industrial sacks and packaging solutions.

## Features

- Premium woven PP sacks
- FIBC Jumbo bags
- BOPP bags
- Custom printing and branding
- Pan-India delivery
- 8+ years of industry experience

## Getting Started

### Prerequisites
- Node.js 18+
- npm or bun

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will start at http://localhost:8080

### Build

```bash
npm run build
```
